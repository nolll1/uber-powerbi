# 🚖 Uber Ride Analytics Dashboard (Power BI)

## 📊 Overview
This project presents an interactive ride analytics dashboard built using Microsoft Power BI to analyze booking trends, revenue, and operational performance for a ride-sharing service.

The dashboard provides insights into trip completion, cancellations, revenue distribution, and customer/driver behavior.

---

## 🚀 Key Features

- 📈 KPI Tracking

- 📅 Time-Based Analysis

- 🚗 Vehicle Type Analysis

- ⭐ Ratings Insights

- 📍 Location Insights

---

## 🧠 Data Modeling

- Designed a **star schema** with fact and dimension tables
- Fact table: Ride transactions (bookings, revenue, distance)
- Dimensions:
  - Date
  - Vehicle Type
  - Location

---

## ⚙️ Tech Stack

- Microsoft Power BI
- DAX (for calculated measures and KPIs)
- Data Modeling (relationships, schema design)
